# 胡氏家族世系圖 (Hu Clan Family Tree)

Interactive family tree scroll for the Hu (胡) clan.

Open `index.html` in any browser to view the interactive genealogy chart.

## Format

- Single-page HTML
- No external dependencies
- Traditional Chinese vertical text layout
- Searchable names
- Color-coded relationship markers

## Branches

- 叟和公
- 旺忠公
- 清和公
- 维显公、维达公、维富公、维有公、维源公
- 五世同堂
